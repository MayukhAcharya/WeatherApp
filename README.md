How you can use the app:


```bash
  git clone https://github.com/MayukhAcharya/News.git
```
```bash
  npm install
```

```bash
  npm start
```
(only for Mac Users)
```bash
  yarn start
```

or if it is not working

```bash
  npx expo start --tunnel
```
